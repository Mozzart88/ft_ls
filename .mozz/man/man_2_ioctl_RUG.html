IOCTL (2) Руководство по системным вызовам IOCTL (2)

ИМЯ

ioctl -
устройство управления

СИНТАКСИС

#include <sys/ioctl.h>

int
ioctl ( int fildes , unsigned long request , ... );

ОПИСАНИЕ

Функция ioctl () манипулирует базовыми параметрами устройства специальных файлов. В частности, многие рабочие характеристики символьных специальных файлов (например, терминалов) могут управляться ioctl запросами (). Аргумент fildes должен быть дескриптором открытого файла.

Запрос ioctl закодировал в нем, является ли аргумент параметром «in» или «out», и размер аргумента argp в байтах. Макросы и определения, используемые при указании запроса ioctl, находятся в файле « sys / ioctl.h» .

ВОЗВРАЩАЕМЫЕ ЦЕННОСТИ

Если произошла ошибка, возвращается значение -1 и значение errno устанавливается для указания ошибки.

ОШИБКИ

ioctl () потерпит неудачу, если:
[ EBADF ]
fildes не является допустимым дескриптором.
[ EINVAL ]
Запрос или argp недействительны.
[ ENOTTY ]
fildes не связан с символом специального устройства.
[ ENOTTY ]
Указанный запрос не применяется к виду объекта, на который дескриптор ссылается.
